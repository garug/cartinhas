## Build Setup

> Intellij

* Edit Configurations
* + Add new Configuration (Alt+Insert) > maven
* `Command line:` spring-boot:run


> VS Code

``` bash
# install dependencies
* npm install 

# serve with hot reload at http://localhost:4200/
* ng serve --open
```
